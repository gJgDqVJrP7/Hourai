# 航空機
## 追加される航空機の種別
### 雷撃機
- 航空魚雷を2発まで搭載可能。
- その他の武装は搭載不可能。
- 雷撃機は1ブロックでも破壊された場合、<撃墜>とする。
### 戦闘爆撃機
- 特攻機でも雷撃機でもない機体。
- 特攻機(ミサイル)を2機まで搭載可能。
- 航空魚雷は搭載不可能。
- 撃破重量は、500B未満の機体は100B、500B以上1000B未満の機体は200B、1000B以上1500B未満の機体は400B、1500B以上2000B未満の機体は600B、2000B以上3000B未満の機体は800B、3000B以上の機体は1200Bとする。
- 100B以上の機体については、羅針盤が破壊された場合もしくは羅針盤と繋がっている総ブロック数が撃破重量を下回った場合、その機体を<撃墜>とする。
- 100B未満の機体については、1ブロックでも破壊された場合<撃墜>とする。
# 航空魚雷
- 雷撃機のみに搭載可能。
- ルートブロックは必要ない。
- 幅1、高さ3、総ブロック数20以下の魚雷に限り、着艦済みの雷撃機に搭載できる。
  - 装填後、当該雷撃機に搭載された魚雷が3発以上になってはいけない。
## 命中判定
- 深度1に着弾する。
- 航空魚雷は、着弾目標に対して自機の全長の「誤差」が発生する。全長が21m以下の場合は、誤差が21になる。
- 航空魚雷は、航空魚雷のいずれかのブロックの進行方向側、船体のあたり判定を有するブロックに接する位置、或いはそこから1m奥に設置することができる。
- 進行経路にそのような位置がない場合、航空魚雷は命中しなかったと判断され、効果を及ぼさないような場所に廃棄する。
# 形状
- 雷撃機及び戦闘爆撃機は、全長+全幅が艦の全長の1/16以上であること。
- 雷撃機及び戦闘爆撃機は、全長が艦の全長未満であること。
- 雷撃機及び戦闘爆撃機は、全高が全長未満であり、且つ全幅未満であること。
- 戦闘爆撃機は、全長+全幅が艦の全長の1/4以下であること。
- 雷撃機は、全長及び全幅が搭載する航空魚雷より大きいこと。
- これらの条件は翼を展開した状態で満たすこと。
